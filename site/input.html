Inputs



To apply your first input copy the html that you want from down below for example let's pick the default input 

                
                    
<div class="tonada-inputs">

                    
    <input class="tonada-input" type="text" name="email" placeholder="Email">

                    
</div>

then call Tonada on this element you can find it with this name Tonada an see the magic happen 

                
                    
document.querySelectorAll('.tonada-inputs').forEach(element => {

                        
    const input = Tonada.create('input', element);

                    
});
